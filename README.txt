@NNMWARE CMF @ 2012-2020

Django-based Content Management Framework

contains base for:

[Core]
    Follows
    Actions
    Tags
    Docs
    Images
    User Notices
    User Messages
    Flat Comments
    Thread Comments
    Thread Categories

[Address]
    Country
    Region
    City

[Money]
    Transaction
    Currency
    Exchange Rate

[..other]
    Profiles
    Topics
    Boards
    Articles
    OEmbed Video Parsing and Storage

[Booking]
    Booking application base 

[Market]
    Market application base 

[Dossier]
    Dossier base 

[Realty]
    Realty application base

Required django(current), Pillow, unidecode, Beautiful Soup.
Working with OpenStreetMap and his geotargeting.
